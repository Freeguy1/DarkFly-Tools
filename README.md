# Install Guide
How to install This project.
* **install on Linux (as superuser)**

```
#apt-get install git
#git clone https://github.com/HackProX/DarkFly-Tools
#cd DarkFly-Tools
#chmod +x install.py
#python3 install.py
#DarkFly
```

* **install on Termux android**

```
$pkg install git python -y
$git clone https://github.com/HackProX/DarkFly-Tools
$cd DarkFly-Tools
$chmod +x install.py
$python main.py
$DarkFly
```
